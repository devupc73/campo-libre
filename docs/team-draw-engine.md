# Motor de Sorteo

## Objetivo

Generar equipos balanceados para un partido utilizando jugadores habilitados.

## Reglas iniciales

- Solo participan jugadores con pago confirmado.
- Los suplentes quedan fuera si se supera la capacidad.
- El sorteo es aleatorio en la primera version.

## Evolucion futura

- Balance por posicion.
- Balance por nivel.
- Historial de enfrentamientos.
- IA para equilibrio competitivo.
